# ☕ CUPLY — Copos personalizados com carinho

Projeto de uma landing page moderna e responsiva para uma loja de copos e canecas personalizadas.

A aplicação foi desenvolvida com foco em experiência do usuário, design acolhedor e navegação intuitiva, permitindo visualizar produtos, personalizações e realizar pedidos de forma simples e prática.

## ✨ Funcionalidades

* Catálogo dinâmico por categorias
* Modal interativo para visualização ampliada dos produtos
* Galeria com múltiplas imagens por produto
* Carrinho de compras com múltiplas etapas
* Integração com API ViaCEP para preenchimento automático de endereço
* Layout totalmente responsivo
* Animações e interações visuais
* Integração com WhatsApp para contato e personalização

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* jQuery
* Bootstrap
* ViaCEP API

## 🎨 Objetivo do projeto

Criar uma experiência visual moderna e humanizada para uma marca de personalizados, unindo estética delicada, usabilidade e interações dinâmicas no frontend.

## 📱 Responsividade

O projeto foi pensado para funcionar de forma fluida em:

* Desktop
* Tablets
* Smartphones

## 💖 Sobre o projeto

A CUPLY nasceu com a proposta de transformar momentos especiais em presentes únicos e personalizados, trazendo identidade visual acolhedora e uma experiência mais próxima do cliente.
